ğŸ§  MCA + Triangle Area Technique

MCA measures correlations between selected features. Triangle-Area maps 3 features into geometric space and detects abnormalities based on area changes.

  • Extract 3 important features
  • Map them as triangle coordinates
  • Compute triangle area for each record
  • Compare with normal traffic patterns

# Extracting important DDoS related features
features = ['src_bytes', 'dst_bytes', 'count']
X = data[features].values

# Function to calculate Triangle Area
def triangle_area(a, b, c):
    return abs(a*(b-c) + b*(c-a) + c*(a-b)) / 2

areas = []
for row in X:
    a, b, c = row
    areas.append(triangle_area(a, b, c))

# Adding area to dataset
data['area'] = areas

# Train-Test Split
X_train, X_test, y_train, y_test = train_test_split(
    data[['area']], data['label'], test_size=0.2
)

# Classification Model
model = RandomForestClassifier()
model.fit(X_train, y_train)

predictions = model.predict(X_test)
